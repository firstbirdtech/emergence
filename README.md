# eMERGEnce

**!!!THIS PROJECT IS UNDER ACTIVE DEVELOPMENT AND NOT READY FOR USAGE YET !!!**

eMERGEnce is a bot that helps you to get rid of all your emerged pull requests automatically on configurable conditions.
